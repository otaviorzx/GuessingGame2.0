- Regras:

  • Para iniciar a partida, coloque um número e confirme, você perceberá pelas imagens e textos se o seu número é maior ou menor.
  • Caso queira reiniciar a partida, clique no botão "reiniciar partida".


- Como o jogo funciona?

  • Quando você clicar em "inserir número", automaticamente o jogo inicia criando um número aleatório entre 1 a 100.
  • Após isso, o sistema lê se o número que você digitou é maior ou menor que o número randomicamente gerado.
  • A imagem da direita do site indica caso o número é maior ou menor.
  • Caso seja menor, uma seta vermelha para baixo é indicada, caso seja maior, uma seta verde para cima aparece. Mas caso você não seja bom em reconhecer setas, você também pode ler o texto que aparece dentro do quadrado branco, logo abaixo.
